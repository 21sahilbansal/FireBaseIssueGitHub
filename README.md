# FireBaseIssueGitHub
This App shows list of firebaseissues and comments associated with the issues
* **It uses** 
* **Retrofit**
* **GSON**
* **Dagger dependency injection**
* **Databinding**
* **Kotlin**
* **Shimmer Effect**
* **MVVM**
* **Observer Pattern**
* **LiveData**
